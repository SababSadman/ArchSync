# ArchSync Feature List & Roadmap

This file tracks the features of ArchSync, categorized by module. Each feature includes its target phase, priority, and current implementation status.

## 📋 Project Workspace
| Feature | Description | Phase | Priority | Status |
| :--- | :--- | :---: | :---: | :--- |
| **Project dashboard** | Overview of status, deadlines, team activity, and recent file uploads. | Phase 1 | High | ✅ Completed |
| **Phase-based structure** | Schematic, design development, construction docs, and closeout phases. | Phase 1 | High | ✅ Completed |
| **Project templates** | Pre-built structures for residential, commercial, and interior projects. | Phase 2 | Medium | ⏳ Not Started |
| **Milestone tracker** | Key dates tied to deliverables with automatic team alerts. | Phase 1 | High | 🚧 In Progress |
| **Multi-project portfolio view** | Studio-level view of all active projects in a single dashboard. | Phase 1 | High | ✅ Completed |

## 📁 File & CAD Management
| Feature | Description | Phase | Priority | Status |
| :--- | :--- | :---: | :---: | :--- |
| **Multi-format upload** | DWG, DXF, RVT, SKP, IFC, PDF, PNG, PSD, and MP4 flythrough videos. | Phase 1 | High | ✅ Completed |
| **In-browser 2D/3D CAD preview** | View CAD drawings and 3D models without opening native software. | Phase 2 | High | 🚧 In Progress |
| **Smart file naming** | AI suggests standardised file names based on project phase and type. | Phase 3 | Low | ⏳ Not Started |
| **BIM model viewer** | Navigate IFC/Revit models in browser, isolate floors and systems. | Phase 2 | Medium | ⏳ Not Started |
| **File grid view** | Card grid showing thumbnails, type icons, file name, size, uploader. | Phase 1 | High | ✅ Completed |

## 🔄 Version History
| Feature | Description | Phase | Priority | Status |
| :--- | :--- | :---: | :---: | :--- |
| **Auto-versioning on upload** | Each upload creates a timestamped version, all previous versions preserved. | Phase 1 | High | ✅ Completed |
| **Named releases** | Pin versions as official milestones: "Schematic Design V1 — Issued for Review". | Phase 2 | Medium | ⏳ Not Started |
| **Side-by-side comparison** | Overlay old and new renders with a slider to reveal differences. | Phase 2 | Medium | ⏳ Not Started |
| **AI-generated changelog** | Auto-writes what changed: "North facade cladding revised; window height reduced". | Phase 3 | High | ⏳ Not Started |
| **One-click restore** | Roll back any file to any past version with full audit trail. | Phase 1 | High | ⏳ Not Started |

| Feature | Description | Phase | Priority | Status |
